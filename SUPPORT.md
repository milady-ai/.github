## 📖 Common Starting Points

**Setting up a MiladyAI elizaOS agent:**
```bash
git clone https://github.com/milady-ai/milady-agent
cd milady-agent
pnpm install
cp .env.example .env
pnpm start --character="characters/milady.json"
```

**elizaOS quick reference:**
- [elizaOS Character File format](https://eliza.how/docs/core/characterfile)

- [elizaOS supported clients](https://eliza.how/docs/clients/overview)

## 🐛 Reporting Bugs

- Search existing [GitHub Issues](https://github.com/milady-ai) before filing
- If it's an elizaOS core bug, report it to [elizaOS issues](https://github.com/elizaOS/eliza/issues)
- For security issues, see our [Security Policy](SECURITY.md)

---

## 💬 Response Times

We're a small open-source team. Response times are best-effort:

- **Discord:** Usually within 24–48 hours
- **GitHub Issues:** 3–5 business days
- **Security reports:** Within 48 hours
