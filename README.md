## About MiladyAI

**MiladyAI** builds open-source AI agents powered by **[elizaOS](https://github.com/elizaOS/eliza)** — the leading multi-agent AI framework. Our agents operate across Twitter, Discord, Telegram, and beyond, blending aesthetic sensibility with cutting-edge AI capability.

### Built on elizaOS

Every MiladyAI agent runs on the elizaOS runtime. We contribute back to the elizaOS ecosystem through:

- 🔌 **elizaOS plugins** — reusable Actions, Providers, and Evaluators
- 🎭 **Character files** — carefully crafted elizaOS agent personas
- 🧠 **Knowledge bases** — curated lore and context for elizaOS memory (RAG)
- 🤖 **Multi-agent setups** — elizaOS agent networks with specialized roles

> elizaOS docs: [eliza.how](https://eliza.how) · [elizaOS GitHub](https://github.com/elizaOS/eliza)
